# 🦸‍♂️ Hero Mom (가제) - Game Design Document

## 1. 게임 개요 (Game Overview)
- **엘리베이터 스피치:** "영웅이 되고 싶은, 하지만 내 마음대로 통제되지 않는 아들을 뒷바라지하는 방치형 크래프팅 시뮬레이션"
- **장르:** 방치형(Idle) / 크래프팅(Crafting) / 관찰 시뮬레이션
- **플랫폼:** Web (HTML5 / JS 프로토타입)
- **목표:** 최종 보스나 엔딩은 없는 무한 성장형. 유저는 아들이 더 강한 몬스터를 잡고 진귀한 전리품을 가져오도록 끝없이 장비와 환경을 업그레이드한다.

## 2. 핵심 플레이 루프 (Core Loop)
이 게임은 유저(부모)와 시스템(아들)이 서로 독립적으로 움직이며 상호작용하는 구조를 가집니다.

1. **부모의 일 (유저 컨트롤):** 
   - 방치 보상(부업)으로 기본 자금 확보
   - 농사(시간 기반)로 식재료 획득 -> 요리/연금술
   - 대장간(크래프팅/뽑기)에서 장비 제작 및 강화
   - 집안 환경(허수아비, 책, 침대) 업그레이드
2. **아들의 일 (AI 자동 제어):**
   - **[집 안]** 체력/허기/경험치 회복 로직 (밥 먹기, 자기, 훈련하기, 책 읽기)
   - **[조건 달성]** 허기 & 체력이 80% 이상이고 가장 좋은 장비를 착용하면 '외출(모험)' 시작
3. **외출과 복귀 (방치 시간):**
   - 외출 시간: 프로토타입 기준 3~5분
   - 외출 중: 랜덤하게 '우편(상태, 경험, 텍스트)' 발송, 마을 NPC를 통해 아들의 소문(로그) 확인 가능
   - 복귀: 모험 결과 브리핑(대사) 및 전리품(사체, 약초, 강화 재료, 골드, 씨앗 등) 획득
4. **반복:** 얻은 전리품으로 다시 부모가 장비와 환경을 업그레이드.

## 3. 세부 메커니즘 (Mechanics)

### 3.1. 화면 구성 (Camera View)
- **단면도 뷰 (Fallout Shelter Style):** 화면 중앙에 집의 단면(침대, 식탁, 허수아비, 책상 등)이 보이고, 아들 캐릭터가 집 안을 돌아다니며 각 가구와 상호작용하는 것을 관찰할 수 있음.

### 3.2. 부모의 액션 (UI 기반 상호작용)
- **부업 (Passive Income):** 유저가 '바느질' 등의 버튼을 누르거나(클릭당 소량 골드), 가만히 둬도(시간당 골드) 기본 자금이 모임.
- **농사 & 요리:**
  - 밭에 씨앗을 심고 일정 시간 후 수확.
  - 수확한 작물로 '요리(허기 회복용)' 또는 '약물(스탯 버프용)' 제작. 완성된 요리는 식탁에 올려둠.
- **대장간 (장비):** 골드와 전리품을 소모해 무기/방어구를 제작(랜덤 스탯 부여)하거나 강화.
- **상점 (가구/환경):** 책(랜덤 스탯 상승), 허수아비(경험치 획득량 증가) 구매 및 배치.
- **마을 소문 (게시판):** 아들이 외출 중일 때, NPC들이 "당신 아들이 슬라임을 잡고 있더라" 같은 텍스트 로그를 띄워 지루함을 달래줌.

### 3.3. 아들의 상태와 AI (State Machine)
- **스탯 (Stats):** 체력(HP), 허기(Hunger), 경험치(EXP)/레벨, 근력(STR), 지능(INT) 등
- **행동 패턴 (Behavior Tree):**
  - `if (외출 중)`: 타이머 작동, 주기적 우편 발송, 타이머 종료 시 복귀 및 전리품 계산.
  - `else if (체력 < 50%)`: 침대/의자로 이동하여 휴식.
  - `else if (허기 < 50% && 식탁에 음식 있음)`: 식탁으로 이동하여 식사.
  - `else if (부상 상태)`: 치료 대기.
  - `else`: 허수아비 훈련(EXP 증가) 또는 책상 독서(랜덤 스탯 증가).
  - `if (체력 >= 80% && 허기 >= 80%)`: 인벤토리 내 최고 등급 장비 자동 장착 후 외출 실행.

## 4. 메타 시스템 (Meta Game)
- **재화 흐름:** 
  - (획득) 부업(기본 골드), 아들의 전리품(희귀 재료, 골드)
  - (소모) 씨앗 구매, 장비 제작/강화, 가구 업그레이드
- **성장 곡선:** 아들의 레벨이 오를수록 외출 시 마주치는 몬스터 수준(가상의 난이도 티어)이 상승하고, 가져오는 전리품의 등급과 양이 증가함.

## 5. 초기 설정 (Initial State)
- **자금:** 1,000 골드, 기본 씨앗 5개
- **집 환경:** 허름한 침대, 빈 식탁, 나무 몽둥이 1개
- **아들 상태:** Lv.1, 체력/허기 100% (시작하자마자 곧 첫 외출을 나감)
